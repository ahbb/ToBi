## Getting Started

### Prerequisites
Install required packages
```sh
  npm install requirements.txt
```

<!-- RUNNING BOT -->
## Running the bot

```sh
cd toilet-api
python -m uvicorn app:app --reload
```

```sh
cd toilet-bot
python bot.py
```
